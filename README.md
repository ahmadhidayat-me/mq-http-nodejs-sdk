# MQ Nodejs HTTP SDK

Alyun MQ Documents: http://www.aliyun.com/product/ons

Aliyun MQ Console: https://ons.console.aliyun.com

## Installation

Add dependency `@aliyunmq/mq-http-sdk`, get the latest version from [npmjs](https://www.npmjs.com/)
```bash
npm install --save
```
*Note: nodejs >= 7.6.0*
